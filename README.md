# Video Game Analysis ETL Pipeline 

## Introduction
This project utilizes the RAWG database to analyze PC games released on Steam in 2025, revealing game release patterns and identifying the games with the highest average playtime, highest average rating, and the greatest number of user ratings. The data was retrieved through the RAWG REST API, cleaned and transformed using Python, then loaded into a SQL database. SQL reports were created to support the Tableau dashboard.

## Data Source
RAWG Video Games Database
